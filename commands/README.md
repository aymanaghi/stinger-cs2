# CS2 Ready-to-Use Commands

Copy any `.cfg` file content and **paste directly into CS2 console** — or add to your `autoexec.cfg`.

## How to Use
1. Open CS2
2. Enable Developer Console (`Settings > Game > Enable Developer Console`)
3. Press `~` to open console
4. Paste commands (or load via `exec filename.cfg`)

## Files
- `display.cfg` → Resolution & brightness  
- `crosshair.cfg` → Simple green dot crosshair  
- `binds.cfg` → Jump on mouse wheel + quick comms  
- `viewmodel.cfg` → Clean weapon POV  
- `performance.cfg` → Max FPS, low latency  

> ✨ Pro tip: Combine them into one `autoexec.cfg` for full setup!
